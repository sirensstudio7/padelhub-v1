# PadelHub PWA

## Project info

A React PWA built with Vite, TypeScript, and shadcn-ui.

## How to run locally

Requirements: Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
